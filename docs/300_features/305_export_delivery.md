# 305 导出与交付

## 1. 功能目标

把编辑结果稳定输出为外部可直接使用的文件，而不是只停留在应用内部。

## 2. 主要场景

- 导出 SRT 交给剪辑或分发流程
- 导出 FCPXML 进入 Final Cut Pro
- 在固定目录中查找输出结果

## 3. 功能范围

包含：

- 生成 SRT
- 生成 FCPXML
- 选择输出位置
- 与 FCP 工作流衔接

不包含：

- 多格式批量导出中心
- 云端发布

## 4. 关键规则

- 导出入口必须明确
- 输出路径必须可预测
- 空字幕处理策略必须一致
- 输出文件应与当前编辑结果保持同步

## 5. 输出要求

- SRT 适合通用字幕交付
- FCPXML 适合剪辑软件工作流
- 当目录中存在 FCP 目标上下文时，应尽量减少用户额外操作

## 6. 验收标准

- 可得到 SRT 文件
- 可得到 FCPXML 文件
- 用户知道文件保存到了哪里
- FCP 工作流下交付结果可继续使用
