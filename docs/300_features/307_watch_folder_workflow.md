# 307 目录监听工作流

## 1. 功能目标

为 Final Cut Pro 等工作流提供一个“放入素材后自动处理”的旁路能力，减少重复操作。

## 2. 主要场景

- 用户把 FCP 导出的新音频放进指定目录
- 系统发现新文件后自动开始处理
- 用户选择全自动导出，或人工校对后再导出

## 3. 功能范围

包含：

- 选择监听目录
- 启动 / 停止监听
- 发现新素材
- 触发转写与可选校对
- 自动导出或载入编辑器
- 通知处理状态

不包含：

- 大规模队列调度
- 后台服务常驻管理
- 团队共享监听任务

## 4. 关键规则

- 监听必须可显式开启和停止
- 监听不是默认主流程
- 应避免重复处理同一素材
- 必须区分“自动完成”与“等待人工校对”两种结果

## 5. 结果模式

### 5.1 自动模式

- 自动转写
- 可选自动校对
- 自动导出 FCPXML

### 5.2 人工校对模式

- 转写完成后载入编辑器
- 用户手动确认后再导出

## 6. 验收标准

- 能检测到新素材
- 不会对同一文件无限重复处理
- 两种模式的最终结果符合预期
- 用户能看到当前处理状态
