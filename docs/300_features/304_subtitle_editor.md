# 304 字幕编辑工作台

## 1. 功能目标

让用户围绕“文本、入点、出点”高效完成字幕微调。

## 2. 主要场景

- 修改识别文本
- 调整某句字幕出现和消失时间
- 补一条漏掉的字幕
- 合并两条过碎的字幕
- 一边播放一边校时间

## 3. 功能范围

包含：

- 字幕列表浏览
- 当前字幕高亮
- 文本编辑
- 入点编辑
- 出点编辑
- 插入前一条 / 后一条
- 合并下一条
- 音频播放联动

不包含：

- 多轨编辑
- 复杂时间轴
- 批量样式编辑

## 4. 关键规则

- 当前选中字幕必须清楚
- 文本、入点、出点是主操作
- 插入与合并保留，但应是轻量操作
- 空字幕允许存在，用于对齐或补位
- 删除不是核心路径，不应成为主要视觉动作
- `Space` 是主操作键：播放时按下会暂停并进入编辑，编辑时按下会退出编辑并继续播放
- 中文输入法正在组词（IME marked text）时，编辑状态下的 `Space` 必须保留给输入法选词，不触发播放恢复
- 编辑状态下 `Shift + Space` 输入空格，`Esc` 退出编辑
- `Tab / Shift + Tab` 在开始时间、结束时间、字幕文本之间切换
- `J / K / L` 负责后退、暂停取时、循环调速播放
- 快捷键说明必须能在编辑页右侧直接查看，也必须能从全局菜单重新打开

## 5. 页面要求

- 列表可快速浏览多条字幕
- 当前条目有明确反馈
- 编辑状态与浏览状态切换自然
- 播放控制与当前时间可辅助校点
- 设置入口不侵入编辑主体

## 6. 验收标准

- 用户可逐条修改文本、入点、出点
- 插入前后字幕后，用户能继续编辑新条目
- 合并后时间和文本结果符合预期
- 音频模式下可结合播放进行校准
- 播放与编辑可只靠键盘主路径完成，不依赖频繁点鼠标
- 用户能在工作台内看到精简快捷键说明，并可打开完整说明
- 完整快捷键说明中 `Tab / Shift + Tab` 不应被截断或错误换行
