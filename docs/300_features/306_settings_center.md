# 306 设置中心

## 1. 功能目标

把所有配置集中到独立页面处理，不让主页面和编辑页面承担配置负担。

## 2. 主要场景

- 选择转写引擎与语言
- 配置 AI 校对
- 调整字幕样式
- 设置导出参数
- 指定目录监听路径
- 测试当前配置可用性

## 3. 功能范围

包含：

- 转写设置
- 校对设置
- 字幕样式设置
- 导出设置
- 目录监听设置
- 可用性测试入口（内置测试音频 / 样本文本）

不包含：

- 复杂账号系统
- 多用户配置同步

## 4. 关键规则

- 设置页独立存在
- 保存后必须生效
- 可选云能力不应阻塞默认使用
- 测试能力用于验证配置，不应替代正式流程验收
- 主窗口只保留轻量设置入口，不在首页和编辑页分散堆配置项
- Whisper 设置中必须能看到模型下载状态，并优先使用国内可访问镜像

## 5. 配置分组

- 转写引擎与语言
- Whisper 模型下载与云端 ASR 参数
- AI 校对开关、服务商、Base URL、模型与提示词
- 字幕样式
- 导出格式、输出目录、自动覆盖与导出到 FCP
- 目录监听与人工校对模式
- 通用设置（界面语言、菜单栏图标等）

## 6. 验收标准

- 用户能独立完成配置与保存
- 重新进入应用后配置可恢复
- 转写 / 校对 / 导出行为能反映设置结果
- 开启“导出到 FCP”后，导出流程会在生成 FCPXML 后尝试打开 Final Cut Pro 导入文件
