# 306 设置中心

## 1. 功能目标

把所有配置集中到独立页面处理，不让主页面和编辑页面承担配置负担。

## 2. 主要场景

- 选择转写引擎与语言（含本地 FunASR、韩语）
- 配置 AI 校对
- 调整字幕样式
- 设置导出参数
- 指定目录监听路径
- 测试当前配置可用性

## 3. 功能范围

包含：

- 转写设置
- 校对设置
- 字幕样式设置
- 导出设置
- 目录监听设置
- 可用性测试入口（内置测试音频 / 样本文本）

不包含：

- 复杂账号系统
- 多用户配置同步

## 4. 关键规则

- 设置页独立存在
- 保存后必须生效
- 可选云能力不应阻塞默认使用
- 新安装默认配置必须低打扰：菜单栏图标开启、界面语言简体中文、转写引擎本地 FunASR（老用户已有配置不强制改写）、识别语言中文、校对关闭、基本样式横屏内白外黑、导出格式 SRT + FCPXML、导出到 FCP 关闭、自动覆盖关闭、保存位置与原文件同目录、目录监听关闭
- 本地 FunASR / Whisper / Apple 语音：各自在「第一次正式转写」弹一次说明（本地适合草稿；要更准建议云端）；设置页验证不弹
- 转写结果剥句末收尾标点（保留问号/感叹号）；校对结果与导入 SRT 不剥
- 测试能力用于验证配置，不应替代正式流程验收
- 主窗口只保留轻量设置入口，不在首页和编辑页分散堆配置项
- Whisper 设置中必须能看到模型下载状态，并优先使用国内可访问镜像；**安装包默认不内置 Whisper 权重**，用户按需下载
- FunASR / SenseVoice 为推荐本地方案；设置中必须能看到运行时检测、模型与 VAD 下载状态，并优先使用国内可访问镜像
- “菜单栏图标”开关必须控制实际状态栏入口显隐，不只是保存配置
- 菜单栏图标开启时，主窗口点关闭只隐藏窗口并进入菜单栏常驻模式；正常打开和最小化时仍显示 Dock 图标，并可通过 `Command + Tab` 切回应用

## 5. 配置分组

- 转写引擎与语言
- Whisper / FunASR 模型下载与云端 ASR 参数
- 单条字幕最大字数（默认 24，注入公共分段器，统一作用于所有转写引擎的切句结果，而不是各引擎各解一次）
- 单条字幕最大字数必须放在官方 / 自定义 / 本地三种方案共享区域，不能只在自定义和本地设置中出现
- AI 校对开关、服务商、Base URL、模型与提示词
- 字幕样式
- 导出格式（SRT / FCPXML / SRT + FCPXML）、输出目录、自动覆盖与导出到 FCP
- 目录监听与人工校对模式
- 通用设置（界面语言、菜单栏图标等）
- 菜单栏菜单：显示 SubForge、导入文件、开始/停止监听、导出、设置、退出，菜单项使用系统图标保持可扫读

## 6. 验收标准

- 用户能独立完成配置与保存
- 重新进入应用后配置可恢复
- 转写 / 校对 / 导出行为能反映设置结果
- 开启“导出到 FCP”后，导出流程会在生成 FCPXML 后尝试打开 Final Cut Pro 导入文件
- 开启“菜单栏图标”后状态栏出现 SubForge 入口；关闭后入口隐藏
- 菜单栏图标开启时关闭主窗口后应用继续运行，Dock 图标隐藏；从菜单栏或监听流程重新显示主窗口后，Dock 图标恢复
- “基本样式”会影响 FCPXML 导出的字幕标题样式；横屏默认字号 `56 pt`、位置 `X 0 / Y -467 / Z 0`，竖屏默认字号 `35 pt`、位置 `X 0 / Y -495 / Z 0`
- 用户手动修改字号或 X/Y/Z 后，FCPXML 导出必须使用用户设置值

## 7. 2026-07-19 字幕方案体验优化开发前检查

- 复用 `SettingsPill`、`SettingsListSection`、`SettingsValidationState` 与现有分段配置，不新增第二套状态来源。
- “推荐 / 实验”从方案标题中拆为小型蓝色标签，方案枚举仍是唯一选择状态。
- 转写与 AI 校对页签右侧显示当前配置完整性和最近验证结果；验证结果继续由 `AppSettings` 持久化。
- 购买区只调整间距和按钮层级，不改变 StoreKit 商品、购买或刷新额度流程。
- 字幕最大字数提升到方案共享区域，并确保官方智能字幕返回结果也进入同一公共分段器。
